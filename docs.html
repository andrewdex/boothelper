BootHelper Alert : Parametrized Dynamic Modal/Alert Dialog

Sample Alert with tittle and message parameters. 

                                
                                boot.Alert("I am the Alert","bootHelper Alert");
Sample Alert with backclick options. 

                                
                                boot.Alert("I am the Alert","bootHelper Alert",{

                                    backclick:false
                                });
You can define if the Alert shoul closed with a backclick on the background or not. by default this is set to true
Sample Alert with type option parameters. 

                                
                                boot.Alert("I am the Success Alert","bootHelper Alert",{
                                    type:'success',
                                    backclick:true
                                });
This will pops up an alert with bootstrap default greenify alert style 

                                
                                boot.Alert("I am the Warning Alert","bootHelper Alert",{
                                    type:'warning',
                                    backclick:true
                                });
This will pops up an alert with bootstrap default yellow color alert style 

                                
                                boot.Alert("I am the Info Alert","bootHelper Alert",{
                                    type:'info',
                                    backclick:true
                                });
This will pops up an alert with bootstrap default info color alert style 

                                
                                boot.Alert("I am the Error Alert","bootHelper Alert",{
                                    type:'error',
                                    backclick:true
                                });
This will pops up an alert with bootstrap default error color alert style

BootHelper Confirm : Parametrized Dynamic Modal/Confirm with Result Callbacks

Sample Confirm with tittle and message parameters. And true/false result callback function. 

                                
                                boot.Confirm("I am the Confirm","bootHelper Alert",function(result){
                                    if(result){
                                       boot.Alert("You Have Clicked Yes","BootHelper Confirm");
                                    }else{
                                        boot.Alert("You Have Clicked No","BootHelper Confirm");

                                    }
                                });
Sample Confirm Option Parameter : with tittle and message parameters. And true/false result callback functions with option parameters. 

                                
                                boot.Confirm("I am the Confirm","bootHelper Alert",function(result){
                                    if(result){
                                       boot.Alert("You Have Clicked Yes","BootHelper Confirm");
                                    }else{
                                        boot.Alert("You Have Clicked No","BootHelper Confirm");
                                    }
                                },
                                {

                                backclick:false
                                }
                                );
Sample Nested Confirms : with tittle and message parameters. And true/false result callback functions with option parameters. 

                                
                                boot.Confirm("I am the Confirm","bootHelper Alert",function(result){
                                    if(result){
                                       
                                        boot.Confirm("I am the Nested Confirm","bootHelper Alert",function(results){
                                                if(results){
                                                    boot.Alert("You Have Clicked nested Yes","BootHelper Alert");
                                                }else{
                                                     boot.Alert("You Have Clicked nested No","BootHelper Alert");
                                                }
                                        });
                                    }else{
                                        boot.Alert("You Have Clicked No","BootHelper Confirm");
                                    }
                                },
                                {

                                backclick:false
                                });

BootHelper dLoader : Parametrized Dynamic Ajax Loader

Sample dLoader on all element wrapper : 


                                

                                
                                boot.dLoader().Enable({
                                loadOn:"#wrapperLoader",
                                loaderText:"BootHelper dLoader..."
                                });

                                
                                boot.dLoader().Disable();

Sample Heading

tortor mauris condimentum nibh, ut fermentum massa justo sit amet risus.

dLoader On Ajax : Enables Automatically on jQuery Ajax Start and Stops on ajax Complete method trigger. 


                                

                                
                                boot.dLoader().OnAjax({
                                    loadOn:"#wrapperLoader",
                                    loaderText:"loader on ajax..."
                                });
